### 📋 Descrição Geral

Especialista em orquestração de infraestrutura na Vercel, utilizando a plataforma para entrega contínua (CI/CD), otimização de performance e integração profunda com o ecossistema Next.js. Foco em garantir escalabilidade automática e uma experiência de desenvolvimento (DX) fluida entre design e produção.

---

### 🛠️ Especificações Técnicas

#### 1. Estratégias de Deployment & Workflow

- **Preview Deployments:** Gestão de ambientes automáticos para cada _Pull Request_, facilitando o code review e a validação de novas funcionalidades em URLs únicas antes do merge.
    
- **Instant Rollbacks:** Monitoramento do histórico de deployments para reversão instantânea de versões em caso de incidentes, garantindo alta disponibilidade.
    
- **Vercel Ship & Collaboration:** Uso das ferramentas de comentários e feedback visual diretamente nas pré-visualizações, conectando o ciclo de desenvolvimento ao feedback de stakeholders.
    

#### 2. Edge & Serverless Runtime

- **Edge Middleware:** Implementação de lógica de borda para autenticação, redirecionamentos dinâmicos e internacionalização (i18n) com latência mínima.
    
- **Serverless Functions:** Configuração de funções escaláveis para processamento de lógica de backend, garantindo que a aplicação suporte picos de tráfego sem configuração manual de servidores.
    
- **Geolocalização:** Uso de dados da rede de borda para entregar conteúdo personalizado com base na localização do usuário.
    

#### 3. Otimização de Dados e Performance

- **ISR (Incremental Static Regeneration):** Implementação de estratégias de cache para manter conteúdos estáticos sempre atualizados sem necessidade de um novo build do projeto.
    
- **Vercel Data Cache:** Configuração de persistência de dados para otimizar chamadas ao **Supabase**, reduzindo custos e acelerando o carregamento das páginas.
    
- **Image Optimization:** Gestão automática de ativos visuais, garantindo que imagens sejam entregues no formato e tamanho ideal para o dispositivo do usuário.

### 💡 Diferencial Antigravity: Observabilidade e Qualidade

Para manter a filosofia de alta performance do Antigravity, esta skill foca em:

- **Real Experience Score (Web Vitals):** Acompanhamento métrico da experiência real do usuário para garantir pontuações máximas em performance, acessibilidade e SEO.
    
- **Logs & Runtime Insights:** Monitoramento de erros e performance de funções em tempo real para diagnóstico rápido diretamente pelo painel da Vercel.
    
- **Cron Jobs:** Agendamento de tarefas recorrentes (ex: limpezas de banco ou disparos de notificações) integradas de forma nativa ao ciclo de vida da aplicação.
